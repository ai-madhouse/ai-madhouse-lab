Self-hosted Google Fonts, stored in-repo for offline builds:

- `src/fonts/nunito/` (Nunito variable wght 200-1000, includes latin/latin-ext/cyrillic)
- `src/fonts/ibm-plex-mono/` (IBM Plex Mono 400/500/600)

Each family folder contains:
- `.woff2`/`.ttf` files used by `next/font/local`
- `OFL.txt` (SIL Open Font License 1.1, from Google Fonts)
